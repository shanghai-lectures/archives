Olivier Michel
5 years 13 hours ago
 

Hi,

Please check the following:
1) You need to have an active Internet connection and should be able to reach www.cyberbotics.com from a web browser.
2) If you access Internet through proxies, you should configure the proxy parameters in the Webots Preferences the same way they are configured in your web browser (i.e., proxy, proxy user name and proxy password).
3) You should ensure no firewall is blocking Webots to connect to the Internet.

Please let me know if that helps.

-Olivier

Olivier Michel
4 years 12 months ago
 

Hi Magali,

Your proxy configuration is available in the preferences of your web browser. With Firefox, it's in Tools/Options (or Preferences), then Advanced/Network/Connection/Settings. If a proxy is used here, please use the same in Webots (proxy URL, username and password if any). Also, ensure that no firewall is preventing Webots to connect to the web.

Let me know if that helps.

-Olivier
