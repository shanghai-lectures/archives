  • [1]
Olivier Michel
4 years 11 months ago
 

Hi Pablo,

Apparently you need to compile also the "mini_dog_controller". This simulation has actually two controllers: one supervisor named "supervisor_controller.cpp" that you could compile and run successfully and a robot controller named "mini_dog_controller" that you didn't compiled (and hence Webots cannot execute it and it will use the void controller instead). To compile it, simply open the file "mini_dog_controller.cpp" as you did and press the "Build" button. Then, revert the simulation and everything should work.

Please let me know if that helps.

-Olivier
