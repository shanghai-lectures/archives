When making a movie using Webots (as required for certain tasks in exercise 2), make sure you don't save the movie file to a directory path containing spaces in any directory name or the file name (e.g. /Users/johndoe/ShanghAI Lectures/Movie.avi). Otherwise the file won't be correctly generated (at least it didn't here on Mac OS X).
